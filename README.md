# opbanner

opbanner is a customizable command-line tool that displays a dynamic, colorful ASCII art banner in your terminal when you start a new shell session (Bash or Zsh). It features a gradient color scheme, customizable messages, and support for custom ASCII art banners.

## *Installation* 

### **From PyPI**
```bash
pip install opbanner
```

**Usage:**
```bash
opbanner
```

### **From Git**

**Clone Repository:** 
```bash
git clone https://github.com/ghostescript/opbanner
cd opbanner
```

**Virtual Environment:** 
```bash
python3 -m venv venv
source venv/bin/activate
```

**Prerequisites:**
```bash
python3 -m pip install -r requirements.txt
```

**Usage:**
```bash
python3 setup.py
```
